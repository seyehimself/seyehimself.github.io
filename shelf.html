Web Developer

The job of a web developer is to create a website that is specific to a client’s specifications including overall look and feel, as well as functionality and features. Depending on the company, the actual code writing is often completed by one person (the web developer) and the actual design of the site is completed by another person (the web designer) using programs like Photoshop. That isn’t always the case. Sometimes the whole process is lumped together under the job title of web developer (or web designer).
